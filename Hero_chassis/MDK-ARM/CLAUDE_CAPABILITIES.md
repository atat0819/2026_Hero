# Claude Code 能力说明

## 模型

当前由 **DeepSeek-V4-Pro** 模型驱动，运行在 Claude Agent SDK 框架下。

## 代码相关能力

| 能力 | 说明 |
|------|------|
| 代码阅读 | 支持几乎所有编程语言和文件格式（.c/.h/.py/.js/.ts 等），也可读取 PDF、图片、Jupyter Notebook |
| 代码编辑 | 精确的字符串替换（Edit）、完整文件重写（Write） |
| 代码搜索 | 正则表达式搜索（Grep）+ 文件名通配符匹配（Glob） |
| 命令执行 | 运行 Shell 命令，支持编译、测试、包管理、git 操作等 |

## 版本控制

- Git 状态查看、差异比较、提交、分支管理
- 创建 Pull Request（通过 GitHub CLI）
- 遵循安全的 Git 操作规范（不随意 force push、不跳过 hooks）

## 项目管理

- **TodoWrite** — 创建和跟踪多步骤任务列表，实时更新进度
- **计划模式** — 复杂实施前先设计方案，获取用户批准后再执行

## 子代理系统

| 子代理 | 用途 |
|--------|------|
| Explore | 快速只读搜索，定位代码文件和符号 |
| Plan | 软件架构师代理，设计实施方案 |
| general-purpose | 通用代理，处理复杂多步骤任务 |
| claude-code-guide | 回答 Claude Code CLI / SDK / API 相关问题 |
| security-review | 安全审查 |

## 其他能力

- **网络搜索** — 获取最新技术文档和信息
- **网页抓取** — 读取并分析网页内容
- **记忆系统** — 跨会话持久化用户偏好和项目上下文
- **定时任务** — 设置一次性或周期性提醒/任务
- **技能系统** — 通过 `/skill` 调用专门的工作流
