## 视觉电控通信串口协议

#### 视觉 -> 电控

| 字节位置 | 数据名称 | 类型 | 说明 |
|---------|---------|------|------|
| 0 | head_one | uint8 | 帧头 1 = 0x39 |
| 1 | head_two | uint8 | 帧头 2 = 0x39 |
| 2-5 | pitch_angle | int32 | Pitch 角度（大端，需除以 100 转为度） |
| 6-9 | yaw_angle | int32 | Yaw 角度（大端，需除以 100 转为度） |
| 10 | vision_ready | uint8 | 视觉识别标志（0 或 1） |
| 11 | fire | uint8 | 开火标志（0 或 1） |
| 12 | tail | uint8 | 帧尾 = 0xFF |
| 13-16 | time | uint32 | 视觉端时间戳（大端） |
| 17 | aim_x | uint8 | 目标 X 坐标 |
| 18 | aim_y | uint8 | 目标 Y 坐标 |
- **总计**: 19字节



#### 电控 -> 视觉

| 字节位置 | 数据名称 | 类型 | 说明 |
|---------|---------|------|------|
| 0 | head_one | uint8 | 帧头 1 = 0x39 |
| 1 | head_two | uint8 | 帧头 2 = 0x39 |
| 2-5 | quat_w | float | 四元数 W 分量 |
| 6-9 | quat_x | float | 四元数 X 分量 |
| 10-13 | quat_y | float | 四元数 Y 分量 |
| 14-17 | quat_z | float | 四元数 Z 分量 |
| 18-21 | bullet_rate | float | 弹速 (m/s) |
| 22 | enemy_color | uint8 | 敌方颜色（0x42 我红 / 0x52 我蓝） |
| 23 | vision_mode | uint8 | 视觉模式 |
| 24 | tail | uint8 | 帧尾 = 0xFF |
| 25-28 | time | uint32 | 时间戳 |
- **总计**: 29字节
