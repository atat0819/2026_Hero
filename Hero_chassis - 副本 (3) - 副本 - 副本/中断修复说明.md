# CAN 和 UART 中断无法触发问题 - 修复说明

## 🔴 问题根本原因

### 1. 中断优先级配置错误（最关键）

**问题描述：**
- FreeRTOS 配置：`configLIBRARY_MAX_SYSCALL_INTERRUPT_PRIORITY = 5`
- 所有中断优先级都设置为 **5**（临界值）
- 在 STM32 + FreeRTOS 中，优先级 = `configLIBRARY_MAX_SYSCALL_INTERRUPT_PRIORITY` 的中断行为未定义

**影响：**
- 中断可能根本不会触发
- 即使触发也无法调用 FreeRTOS API（如 `xQueueOverwriteFromISR`）
- 系统可能死锁

**FreeRTOS 中断优先级规则：**
```
优先级数值 0-4   : 不能调用 FreeRTOS API（高优先级，用于紧急中断）
优先级数值 5     : 临界值，行为未定义 ❌
优先级数值 6-15  : 可以调用 FreeRTOS API ✅
```

### 2. CAN 过滤器未配置

**问题描述：**
- CAN 启动了，中断也激活了
- 但是没有配置 CAN 过滤器
- **没有过滤器，CAN 不会接收任何消息**

### 3. UART DMA 循环模式冲突

**问题描述：**
- USART1 配置为 `DMA_CIRCULAR` 模式
- 但代码中尝试手动重启 DMA
- 循环模式下不应该手动重启，会导致冲突

---

## ✅ 已完成的修复

### 修复 1：修改所有中断优先级为 6

**修改文件：**
1. `Core/Src/dma.c` - 所有 DMA 中断优先级：5 → 6
2. `Core/Src/usart.c` - UART1/UART8/USART3/USART6 中断优先级：5 → 6
3. `Core/Src/can.c` - CAN1/CAN2 中断优先级：5 → 6

**修改内容示例：**
```c
// 修改前
HAL_NVIC_SetPriority(USART1_IRQn, 5, 0);  // ❌ 临界值

// 修改后
HAL_NVIC_SetPriority(USART1_IRQn, 6, 0);  // ✅ 可以调用 FreeRTOS API
```

### 修复 2：添加 CAN 过滤器配置

**修改文件：** `Core/Src/main.c`

**添加内容：**
```c
// CAN1 过滤器配置（接收所有消息）
CAN_FilterTypeDef canFilter;
canFilter.FilterBank = 0;
canFilter.FilterMode = CAN_FILTERMODE_IDMASK;
canFilter.FilterScale = CAN_FILTERSCALE_32BIT;
canFilter.FilterIdHigh = 0x0000;
canFilter.FilterIdLow = 0x0000;
canFilter.FilterMaskIdHigh = 0x0000;
canFilter.FilterMaskIdLow = 0x0000;
canFilter.FilterFIFOAssignment = CAN_RX_FIFO0;
canFilter.FilterActivation = ENABLE;
canFilter.SlaveStartFilterBank = 14;

HAL_CAN_ConfigFilter(&hcan1, &canFilter);

// CAN2 过滤器配置
canFilter.FilterBank = 14;  // CAN2 从过滤器组 14 开始
HAL_CAN_ConfigFilter(&hcan2, &canFilter);
```

### 修复 3：移除 UART DMA 手动重启

**修改文件：** `RtosTask/remote_task.cpp`

**修改内容：**
```cpp
// 已注释掉手动重启 DMA 的代码
// 因为 USART1 配置为 DMA_CIRCULAR 模式，会自动循环接收
```

## 🔍 如果问题仍然存在

### 检查清单：

1. **硬件连接**
   - [ ] 遥控器接收器正确连接到 USART1（PB6-TX, PB7-RX）
   - [ ] CAN 收发器正确连接（CAN1: PD0-RX, PD1-TX）
   - [ ] 供电正常

2. **波特率配置**
   - [ ] USART1 波特率：100000，偶校验
   - [ ] CAN 波特率：1Mbps（根据你的配置）

3. **中断使能**
   - [ ] 全局中断已使能（`__enable_irq()`）
   - [ ] NVIC 中断已使能

4. **FreeRTOS 配置**
   - [ ] 确认 `configLIBRARY_MAX_SYSCALL_INTERRUPT_PRIORITY = 5`
   - [ ] 确认所有外设中断优先级 >= 6

5. **DMA 配置**
   - [ ] DMA 时钟已使能
   - [ ] DMA 通道配置正确
